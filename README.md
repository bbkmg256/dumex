# Dumex
Un front-end minimalista para GZDoom.

# Dependencias
- flet
- configparser
- zenity (en linux)

Tener en cuenta que flet requiere de:
- mpv
- mpv-dev

Nota: Para las 2 últimas dependencias es necesario también resolver la biblioteca compartida "libmpv.so.1".
